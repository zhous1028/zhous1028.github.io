深度学习入门——神经网络

Posted on 2022-01-08 Edited on 2022-01-21

感知机的权重由人工设置，而神经网络可以自动从数据中学习到合适的参数。

激活函数h(x)将输入信号的总和转换为输出信号，比如阶跃函数，当输入总和大于0时，输出1，否则输出0。

激活函数是连接感知机和神经网络的桥梁。

“朴素感知机”是指单层网络，指的是激活函数使用了阶跃函数的模型。
“多层感知机”指的是神经网络，即使用sigmoid函数等平滑的激活函数的多层网络。

Activation Function

sigmoid function

$$ h(x)=\frac{1}{1 - e^{-1}} $$

感知机神经元之间流动的是0或1的二元信号，而神经网络中流动的是连续的实数值信号。

激活函数不能使用线性函数，即$ h(x)=cx $的函数，因为如果使用线性函数，那么加深神经网络的层数就没有意义了。

ReLU function

在神经网络的发展史上，sigmoid函数很早就开始被使用了，而最近则主要使用ReLU（Rectified Linear Unit）函数。

$$
h(x)=
\begin{cases}
x \quad x > 0 \\
0 \quad x \leq 0
\end{cases}
$$

三层神经网络的实现

  • 输出层的激活函数用σ()表示，不同于隐藏层的激活函数h()

  • 输出层所用的激活函数，要根据求解问题的性质决定。一般地，回归问题可以使用恒等函数，二元分类问题可以使用sigmoid函数，多元分类问题可以使用softmax函数。

深度学习入门——感知机

Posted on 2022-01-05 Edited on 2022-01-08

感知机：perceptron

感知机接收多个信号，输出一个信号。

机器学习中的学习是指确定合适的参数的过程。

权重和偏置的作用是不同的：
权重表示输入信号的重要性，
偏置表示神经元被激活的容易程度。

异或门：相同得0，相异得1

单层感知机可以表示与门、与非门、或门，无法表示异或门。
单层感知机可以分离线性空间，无法分离非线性空间。

两层感知机可以表示异或门。
理论上两层感知机就可以构建计算机。

gcc 与 Dev-C++ 的关系

gcc 是一套编译器集合。
Dev-C++ 是 Windows 上的一个IDE，gcc是Dev-C++默认使用的编译器。
即，gcc 是内核，Dev-C++是外壳。

Windows10 下安装 gcc、g++

  • 我已经安装了Dev-Cpp
  • 接下来只需将 MinGW64/bin 设置为环境变量

Note：

  • 修改环境变量后，如果受影响的是应用程序，则只需重启应用程序，环境变量的修改就会反映到该程序中，无需重启电脑。
    因此，当我要在cmd中测试gcc是否安装成功时，需要重启cmd，才能看到改变！

  • 如果受影响的是系统服务，则必须重启电脑，因为无法在不重启计算机的情况下重新启动系统服务管理器。

  • 通过 gcc -v 和g++ -v分别测试gcc、g++是否安装成功（记得要重启cmd）

  • 使用gcc或g++编译c程序

  • Linux下生成.out文件（Linux下文件后缀只是提示，是否可执行取决于有无 -x <执行> 权限）
